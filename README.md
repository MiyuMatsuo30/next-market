# 概要
- ユーザーが販売したいアイテムをアプリケーション上で登録、修正、削除ができる
- ログイン機能

## 目的
- Web系の言語で用いる技術の基本を体系的に学ぶ

## 作成サイト(写経+α)
- [URL](https://nextjs-app-beta-topaz.vercel.app/item/create)

### 基本実装
- ユーザー
  - 登録
  - ログイン
- 商品
  - 作成 (ログインユーザのみ)
  - 更新 (ログインユーザ、作成ユーザのみ)
  - 削除 (ログインユーザ、作成ユーザのみ)
